
# eCommerce

Plataforma de comercio electrónico especializada en **productos tecnológicos** como periféricos gaming, hardware de computadora y muebles ergonómicos.

## Características

- **Catálogo dinámico**: Sistema de productos con 17 artículos de diferentes categorías (Periféricos, Hardware, Muebles)
- **Filtrado por categoría**: Usuarios pueden filtrar productos por tipo
- **Paginación**: Visualización de productos por páginas con botón "Ver más"
- **Carrito de compras**: Sistema persistente con localStorage
- **Interfaz moderna**: Diseño visual con gradientes, efectos hover y animaciones suaves
- **Responsive**: Adaptable a dispositivos móviles
- **Valoración y disponibilidad**: Cada producto muestra calificación y estado de stock

## Estructura técnica

- **HTML5**: Semántica y accesibilidad (aria-labels)
- **CSS3**: Variables de color, grid responsive, animaciones
- **JavaScript vanilla**: Gestión de DOM, eventos y persistencia de datos
- **LocalStorage**: Almacenamiento del carrito entre sesiones

## Categorías de productos

- Periféricos (teclados, mouse, auriculares, webcams)
- Hardware (tarjetas gráficas, RAM, gabinetes, fuentes)
- Muebles (sillas gaming)
